# Web Authentication System with Calculator

## 🌐 Live Project
The project is available online:  
https://shapran.site

---

## 📌 About the Project
This is a web application that includes a user authentication system with email verification and an interactive calculator.

The system allows users to register, confirm their account via email code, and log in securely. After authentication, users can access the calculator functionality.

---

## 🚀 Features
- User registration system
- User login system
- Email verification via confirmation code
- Secure authentication logic using PHP
- Interactive calculator built with JavaScript
- Data storage using MySQL (phpMyAdmin)

---

## 🛠️ Technologies Used
- PHP
- JavaScript
- HTML / CSS
- MySQL

---

## ⚙️ Deployment
This project is deployed on a web hosting environment.

### How it works:
- Files are hosted on a server (public web hosting)
- Database is managed via phpMyAdmin
- PHP handles server-side logic and authentication

---

## 📂 Project Structure
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

This project was developed as a personal learning project to practice full-stack web development, including authentication systems and backend integration.
